Open Source Software (OSS) is software for which the source code is open for anyone to view. A related type of open source software is Free and Open Source Software (FOSS) which offers additional freedoms. FOSS is software for which the source code not only affords users the freedom to view the source code but also guarantees users the freedom to build, copy, distribute, change, and improve the software.
