Installing on Linux.

Using your Linux distribution’s preferred package manager is the simplest process for installing Git on Linux.
If you’re on a Debian-based distribution, such as Ubuntu, try apt:
$ sudo apt install git-all
If you’re on Fedora (or any closely-related RPM-based distribution, such as RHEL or CentOS), you can use dnf:
$ sudo dnf install git-all
For more options, there are instructions for installing on many different Linux distributions on the Git website, at https://git-scm.com/download/linux 1 .
