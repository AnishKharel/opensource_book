If you are working in WSL2, Git for Windows might also be useful. Git for Windows provides a native set of tools that bring the full feature set of the Git SCM to Windows as well as bash emulation which can be used to run Git from the command line. To install go to https://gitforwindows.org 3 . Not that Git for Windows will not be useful to you if you are developing in a VM running Ubuntu.
