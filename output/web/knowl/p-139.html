All OSS communities need to have their code hosted in a place where it can be viewed and potentially contributed to. Bitbucket 1 , GitHub 2 , GitLab 3  are three of the most popular a code hosting services used for for collaboration by open source communities. One of the many features these sites offer web hosting, which can facilitate structured collaborative editing of the site. For example, Github Pages 4  affords hosting as a web site for projects managed in GitHUb.
