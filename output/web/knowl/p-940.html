Paragraph

pwd

Print working directory displays the name of the current working directory.

  • e.g. Print the path of the current working directory: pwd

whoami

The whoami prints the userid of the current user.

  • e.g. Print the userid: whoami

ls

List displays basic information about files and directories.

  • e.g. List directories and files in the current directory: ls

  • e.g. list directories and files in the current directory using a long listing: ls -l

  • e.g. list directories and files in home directory: ls ~

touch <file-name>

The touch is commonly used for file creation. Its intended primary function is to update its timestamp, by "touching" it. See man touch for more information on the intended use.

  • e.g. touch newfile.txt

cd <directory-name> or relative position

Change directory to <directory-name>.

  • e.g. change directory to the root directory: cd /

  • e.g. change directory to the home directory: cd ~

  • e.g. change directory to the immediate parent directory: cd ..

mv <old-name> <new-name>

Move (rename) files or directories.

  • e.g. Change the name of old.txt to new.txt: mv old.txt new.txt

rm <file-name>

Remove deletes a file or directory.

  • e.g. Remove the file named junk.txt: rm junk.txt

mkdir <directory-name>

Make directory with parameter name <directory-name>.

  • e.g. Make a new directory with the name newdir: mkdir newdir

rmdir <directory-name>

Remove directory with specified name <directory-name>

  • e.g. Remove (delete) the directory with the name olddir: mkdir olddir
