Let's look at the contents of the files that we created. The cat command stands for concatenate is used to print on the standard output device (usually your screen for our purposes) or to concatenate files. To use cat to see the file contents, type the following one at a time:

cat newfile1.txt
cat newfile2.txt
cat newfile3.txt
You can also use the cat command to concatenate files. For example, if you try:
cat newfile2.txt newfile3.txt > newfile1.txt
cat newfile1.txt
You should see that newfile1.txt instead of being empty, now contains the concatenation of newfile2.txt and newfile3.txt
