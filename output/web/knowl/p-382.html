The most popular editors used by contributors to open source projects include Emacs 1 , Vim 2 , and Visual Studio Code aka VSCode 3 . (Warning, Visual Studio Code is not the same software as as Visual Studio. This clearly causes a lot of confusion!) For novices contributors to open source, VSCode 4  is recommended as a first choice because it is cross-platform and has reasonably shallow learning curve. It also happens to be open-source!
