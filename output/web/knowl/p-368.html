Professional software developers use text editors that are specifically designed to work with source code and are often called code editors or just editors. Code editors typically include many special features for writing and editing code that go well beyond the typical uses such as finding and replacing, undoing, deleting, etc. Some of professional developers prefer to use a single purpose editor while others prefer to use an Integrated Development Environment (IDE) which is a tool that combines commonly used developer tools in a single application.
