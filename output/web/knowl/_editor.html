Your Editor.

Now that your identity is set up, you can configure the default text editor that will be used when Git needs you to type in a message. If not configured, Git uses your systemâ€™s default editor.

If you want to use a different text editor, such as VS Code, you can do the following:

$ git config core.editor "code -n --wait"

Note that on a Windows system, if you want to use a different text editor, you must specify the full path to its executable file. This can be different depending on how your editor is packaged.
