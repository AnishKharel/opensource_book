echo <text>

The echo command displays a line of text and/or the value of a variable. Often used with output redirection.

  • e.g. Print the text 'Hello World!' on the screen: echo 'Hello World!'

  • e.g. Print the value of the USER environment variable on the screen: echo $USER

cat <file-name>

The concatenate prints file contents on the screen after concatenation. Note that with a single file, it just prints that file. It is often used with output redirection.

  • e.g. Print the contents of file.txt on the screen: cat file.txt

  • e.g. Print the contents of the concatenation of file1.txt and file2.txt on the screen: cat file1.txt file2.txt

read <variable-name>

The read command reads a line or variable from the keyboard. It is often used with scripts or input redirection.

  • e.g. Take input from the keyboard and direct it into a variable called MYVAR: read MYVAR

wc <file-name>

The word count command performs a count of lines, words, and bytes for each file.

  • e.g. Report the count of lines, words, and bytes in file.txt: wc file.txt
